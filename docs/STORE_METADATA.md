## Positioning

Canopy Trove should stay positioned as:

- licensed dispensary discovery
- trusted storefront information
- community reviews
- saved favorites and chosen local picks
- verified operator listing management

Canopy Trove should avoid release messaging that implies:

- cannabis ordering
- checkout for cannabis products
- delivery brokering
- illegal sales activity

## Launch Full Description

Canopy Trove helps adults discover licensed dispensaries with a calmer, more trusted experience. Browse storefronts, check the details that matter, save favorites, and read thoughtful community feedback before you choose where to go.

The first Canopy Trove release is built around licensed discovery. Storefront pages bring together hours, website links, ratings, photos, and useful context in one place so the experience feels chosen instead of crowded.

Canopy Trove also includes verified operator tools for licensed storefront teams, with listing management, review follow-up, and visibility controls designed for real businesses. Canopy Trove is not an ordering, checkout, or delivery platform.
