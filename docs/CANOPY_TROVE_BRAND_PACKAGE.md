# Canopy Trove Brand Package

## Core Brand

- Name: `Canopy Trove`
- Positioning: trusted cannabis discovery with premium energy and real personality
- Audience balance: calm enough for legal trust, bright enough for fun

## Recommended Messaging

- Primary tagline: `Where legal cannabis feels chosen.`
- Alternate tagline: `Trusted dispensaries. Better vibes.`
- App Store subtitle: `Dispensaries, deals, reviews`
- Premium alternate subtitle: `Legal cannabis, chosen well`

## Visual System

- Near-black: `#121614`
- Soft ivory: `#F3EBDD`
- Canopy green: `#5E866E`
- Muted gold: `#C8A66A`
- Badge green: `#9FD4AE`

## Icon Concepts

- Canopy Arc: simplest and strongest. Shelter arch over a gold spark.
- Shelter Mark: editorial canopy strokes over a curated gem.
- Crest Icon: premium badge for owner-facing and higher-status moments.

## Moodboard Direction

- Warm premium, not cold luxury
- Shelter, curation, discovery, and celebration
- Avoid map pins, roads, smoke, or literal weed leaves
- The feeling should be selective and social, not exclusive or stiff

## Presentation Outline

1. Brand premise
2. Emotional target
3. Logo directions
4. Color system
5. Consumer voice
6. Owner-premium angle
7. Splash and store-preview direction

## Image Generation Prompts

### Moodboard Prompt

```text
Create a premium brand moodboard for a cannabis discovery app called Canopy Trove. The brand should feel trusted, elevated, welcoming, and slightly celebratory. Use a dark near-black base, soft ivory typography, muted canopy greens, and restrained warm gold accents. Include visual references for shelter, curation, a hidden gem, editorial cannabis retail, premium badge systems, elegant mobile UI fragments, and modern hospitality-inspired texture. Avoid smoke, cannabis leaves, map pins, roads, route icons, or anything that feels like delivery logistics.
```

### Polished Mockup Prompt

```text
Create polished mobile app brand mockups for Canopy Trove. Show an upscale but welcoming cannabis discovery app with refined cards, premium badges, soft gold highlights, and calm green accents. Include one app icon presentation, one homepage/landing mockup, one phone UI mockup, and one owner-facing premium dashboard glimpse. The visual tone should be legal, curated, social, and high-quality without feeling clinical or elitist.
```

### Splash Preview Prompt

```text
Create iOS and Android splash screen previews for Canopy Trove. Use a dark near-black background, a refined canopy emblem, and a small gold discovery spark. The splash screen should feel premium, quiet, and memorable with clean negative space and subtle soft-green glow. Avoid gradients that feel too loud or neon. Keep the mark centered and app-launch ready.
```

## Recommended Next Step

- Use `Canopy Arc` as the default app icon direction.
- Use `Shelter Mark` in the website and pitch deck.
- Keep `Crest Icon` for owner-premium or verification applications.
