# Cannabis Tales Source Library

Last updated: 2026-04-21

Purpose: source pool for **Cannabis Tales**, an educational, sourced story library about cannabis history, policy, culture, justice, and regulation. This is a source/citation plan, not final story copy.

## Editorial Rules

- Write original Canopy Trove story text. Do not copy modern articles.
- Prefer public-domain, U.S. government, CC0, CC-BY, or archive materials with clear rights notes.
- Cite every story with source links.
- Use short quotations only when needed, and avoid long verbatim passages.
- Keep tone educational and adult-rated. Do not provide dosage, growing instructions, buying instructions, medical advice, or product recommendations.
- Label uncertain historical claims as uncertain. Do not let AI invent dates, names, quotes, or motives.

## Current Newspaper Search Pattern

- Use the current `loc.gov` search API, not the retired page-search endpoint: `https://www.loc.gov/search/?fo=json&q=marihuana&fa=partof:chronicling+america&c=10`.
- Use `resource` page links for citations, for example `https://www.loc.gov/resource/{lccn}/{yyyy-mm-dd}/ed-1/?sp={page}`.
- Use text view for OCR inspection when available by adding `&st=text`.
- OCR is not final truth. Always verify against the scanned page image before publishing.
- The LOC migration page says computational users should transition to the LoC.gov API, so any future harvester should be written against `www.loc.gov/search/?fo=json`.
