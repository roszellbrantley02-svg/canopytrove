# Canopy Trove Account Deletion Help

Updated: March 29, 2026

## Delete In The App

Canopy Trove supports in-app account deletion from the Profile area.

To delete an account:

1. Open `Profile`
2. Open the account deletion screen
3. Type `DELETE`
4. Confirm deletion

## What Deletion Does

Deleting an account removes local Canopy Trove data on the device and clears the linked backend profile path used by the app.

## When A Recent Sign-In Is Required

Some authentication providers require a fresh sign-in before the login itself can be removed. If that happens, Canopy Trove should tell you clearly and ask you to sign in again before retrying deletion.

## If Deletion Does Not Finish

If deletion is blocked or incomplete:

- sign in again and retry the in-app flow
- keep any error message shown by the app
- email `askmehere@canopytrove.com` for follow-up
